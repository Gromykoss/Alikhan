# Каналы связи Hermes: варианты для заказчика

**Дата:** 18.07.2026
**Статус:** Документ для заказчика

---

## Что такое Hermes

Hermes Agent — это платформа для автономных AI-агентов, которая поддерживает **несколько каналов связи одновременно**. Бот Alikhan работает через один из них (WhatsApp), но заказчик может выбрать любой.

---

## 1. WhatsApp — основной (рекомендованный)

### Почему WhatsApp
- 100% охват на стройках СНГ и Кыргызстана
- Прорабы и ИТР уже там — не нужно ставить новые приложения
- Групповые чаты — стандартный формат коммуникации на объекте
- Поддержка фото, документов, голосовых сообщений

### Что нужно

| Позиция | Детали |
|---------|--------|
| **Номер телефона** | Отдельная SIM-карта для бота |
| **Группа «Песочница»** | Для тестирования |
| **Группа «Боевая»** | Рабочая группа стройки |
| **Hermes Bridge** | WhatsApp Web API (уже в составе Hermes) |

### Ограничения
- Meta может запросить верификацию при подозрительной активности
- Лимит ~50 сообщений в минуту на номер
- Файлы до 50 MB

### Процесс подключения
1. Заказчик покупает SIM, регистрирует WhatsApp на отдельном телефоне
2. Мы генерируем QR-код на VPS
3. Заказчик сканирует QR через WhatsApp Web (Linked Devices)
4. Бот онлайн — работает 24/7

---

## 2. Telegram — альтернатива

### Почему Telegram
- Самый простой в настройке (не нужен номер телефона)
- Бесплатный Bot API
- Поддержка групп, каналов, inline-кнопок
- Файлы до 2 GB

### Что нужно

| Позиция | Детали |
|---------|--------|
| **Telegram-бот** | Создать через @BotFather (2 минуты) |
| **Токен бота** | Выдаёт BotFather |
| **Группа «Песочница»** | Для тестирования |
| **Группа/канал «Боевая»** | Рабочая |

### Ограничения
- Не все прорабы используют Telegram
- Нет голосовых сообщений через Bot API (только через userbot)
- Бот не может первым писать пользователю

### Процесс подключения
1. Заказчик создаёт бота через @BotFather
2. Передаёт нам токен
3. Мы настраиваем Hermes на приём сообщений
4. Заказчик добавляет бота в группы
5. Готово

## 3. Discord — нишевый

### Почему Discord
- Удобен для распределённых команд
- Каналы по темам (отдельный канал под каждый раздел стройки)
- Threads для обсуждения конкретных вопросов

### Что нужно

| Позиция | Детали |
|---------|--------|
| **Discord-сервер** | Создать бесплатно |
| **Discord-бот** | Через Developer Portal |
| **Токен бота** | Выдаёт Discord |

### Ограничения
- Непривычный для строителей интерфейс
- Нужно ставить приложение
- На стройках почти не используется

---

## 4. Slack — корпоративный

### Почему Slack
- Стандарт в западных EPC/EPCM-компаниях
- Интеграция с Jira, Google Drive, MS Teams
- Аудит и compliance

### Что нужно

| Позиция | Детали |
|---------|--------|
| **Slack Workspace** | Платный (или бесплатный до 10 участников) |
| **Slack App** | Через api.slack.com |
| **Bot Token** | Выдаёт Slack |

### Ограничения
- Платный для команд >10 человек (~$7.25/чел/мес)
- Не используется на стройках СНГ
- Сложная настройка permissions

## Гибридный режим (рекомендован)

Hermes может работать **одновременно через несколько каналов**:

- **WhatsApp** → сбор данных со стройплощадки (основной)
- **Telegram** → уведомления для руководства (отдельный канал)

Например: бот читает чат прорабов в WhatsApp, а ежедневный ЕЖО присылает в Telegram-канал директору.

Настройка гибридного режима — бесплатно, включено в базовую конфигурацию.

## Выбор для Alikhan

**Alikhan использует WhatsApp** — это осознанный выбор:
- ТЗРК Джеруй: все коммуникации в WhatsApp
- Фото со стройплощадки приходят в WhatsApp
- Документы и голосовые — в WhatsApp
- Не нужно переучивать персонал

**Но:** при развёртывании клона для другого объекта заказчик может выбрать Telegram или гибридный режим.

## Чек-лист выбора канала

- [ ] Все ключевые участники используют этот мессенджер? (WhatsApp — почти всегда да)
- [ ] Нужны ли голосовые сообщения от прорабов? (только WhatsApp)
- [ ] Нужен ли отдельный канал для руководства? (Telegram параллельно)
- [ ] Есть ли корпоративные требования? (Slack/Discord — редко)
- [ ] Готовы ли купить отдельный номер для бота? (для WhatsApp — да)

**Рекомендация по умолчанию:** WhatsApp (основной сбор) + Telegram (отчёты руководству).
