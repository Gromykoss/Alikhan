## 2. Что такое Alikhan

**Alikhan** — автономная AI-система управления строительством, работающая прямо в WhatsApp. Это не просто чат-бот: это **автономный агент**, который собирает отчёты от прорабов через мессенджер, парсит сообщения через Grok (xAI), заполняет 14 таблиц ОЖР по ГОСТ РД-11-05-2007 в PostgreSQL и в любой момент формирует готовый ЕЖО. А главное — Alikhan **сам управляет своим кодом**: находит баги, вносит исправления и расширяет функционал через Codex/Grok Build CLI без участия программиста. Клонируется на любой объект за один день.

## 4. Как это работает

```
┌──────────────┐     ┌────────────────┐     ┌─────────────┐     ┌───────────┐
│  WhatsApp    │────▶│  Hermes Bridge │────▶│  Python-бот  │────▶│  Grok AI  │
│  (прорабы)   │     │  (порт :3000)  │     │  main_waha   │     │  (xAI)    │
└──────────────┘     └────────────────┘     └──────┬───────┘     └─────┬─────┘
                                                   │                   │
                                          ┌────────┴────────┐          │
                                          ▼                 ▼          │
                                   ┌───────────┐    ┌───────────┐      │
                                   │ QA-парсер │    │  Router   │      │
                                   │ (VOR-коды)│    │ (команды) │      │
                                   └─────┬─────┘    └───────────┘      │
                                         │                              │
                                         ▼                              │
                              ┌────────────────────┐                    │
                              │    PostgreSQL      │◀───────────────────┘
                              │    14 таблиц ОЖР   │
                              └────────┬───────────┘
                                       │
                    ┌──────────────────┼──────────────────┐
                    ▼                  ▼                  ▼
            ┌────────────┐    ┌────────────┐    ┌────────────┐
            │   ЕЖО      │    │  Снимок    │    │ Аналитика  │
            │ (Excel)    │    │   дня      │    │  по этапам │
            └────────────┘    └────────────┘    └────────────┘
```

> **Поверх этого — контур само-улучшения:**
> Hermes Agent → Codex / Grok Build CLI → автономная правка и расширение кода бота.

### Что происходит в реальном времени:

1. Прораб пишет в WhatsApp-группу: «Армирование плиты Пм-3, 120 м², ось А-Г»
2. QA-парсер (Python + Grok) извлекает: код ВОР, объём, локацию, автора
3. Данные попадают в `ojr_section3_work_log` (Раздел 3 ОЖР)
4. Фото с площадки → Grok Vision → авто-описание → `ojr_photo_log`
5. Вечером — автоопрос «что сделано за день», сводка → `ojr_daily_summary`
6. В любой момент: команда в WhatsApp → готовый ЕЖО в Excel
