## 10 правил

1. **4 блока строго:** 📷 Фото / 📄 Документы / 💬 Активность / 📊 Работы
2. **Фото — не перефразируй.** Описания Grok Vision агрегируются, не реинтерпретируются.
3. **Документы — только факт наличия.** Без интерпретации содержимого.
4. **Активность — о чём говорили.** Темы обсуждений, не пересказ каждого сообщения.
5. **Работы — факт → план.** `код (название) факт=X → план=Y`. Факт всегда перед планом.
6. **Итог — одна строка.** Без оценок и рекомендаций.
7. **Без воды.** Без вводных, заключительных, оценочных фраз.
8. **Без повторов.** Каждый факт ровно в одном блоке.
9. **Пустые блоки.** Выводить с пометкой «не поступали» и т.д.
10. **Русский язык.** Погода на русском.

## Плейсхолдеры

- `{weather}` — строка погоды
- `{photo_block}` — маркированный список фото
- `{doc_block}` — список файлов через запятую
- `{msg_block}` — маркированный список сообщений
- `{poll_block}` — данные ЕЖО/опроса
- `{fact_block}` — QA-факты

## Пример результата

```
📷 Фото: 3 фото. АБК — монтаж сэндвич-панелей. Склад — заливка бетона. Галерея — сборка металлоконструкций.

📄 Документы: Протокол_17.07.pdf, ВОР_корректировка.xlsx

💬 Активность: обсуждение задержки поставки профнастила (20.07), уточнение объёмов 4.2.

📊 Работы: 3 позиции. 3.1.1 (Колонны) факт=12 → план=15; 2.1.5 (Грунт) факт=45.0 → план=50.0.

Итог: Переменная облачность, +27°C. Выполнено 3 позиции.
```
