# EJO Column Map — полная карта (v3, 16.07.2026)

Лист «Ежедневный отчет», столбцы A-W.

| Кол | Excel | Название | Кто | Формула |
|-----|-------|----------|-----|---------|
| A | 1 | Здание | Шаблон | Название объекта |
| C | 3 | Код работы | Шаблон | X.Y.Z |
| E | 5 | Ед. изм. | Шаблон | м³, т, м²... |
| K | 11 | План общий | Шаблон | Контрактный объём |
| L | 12 | План на сутки | Бот | `= v` |
| M | 13 | Факт за сутки | Бот | `= v` |
| N | 14 | % за сутки | Бот | `= v / O * 100`, формат `0.0"%"` |
| **O** | **15** | **План на месяц** | **Пользователь** | **Вручную** |
| P | 16 | Накоп. план (мес) | Бот | `= prev_P + v` |
| Q | 17 | % от мес. плана | Бот | `= P / O` |
| R | 18 | Общий план | Шаблон | Контракт |
| S | 19 | Накоп. факт (общ) | Бот | `= prev_S + v` |
| T | 20 | % от общего | Бот | `= S / R` |
| U | 21 | Остаток | Бот | `= O − P` |

## Row Hiding Logic (generated EJO only)

| Condition | Visible |
|-----------|---------|
| Has work today (L or M) | ✅ |
| O > 0 AND U > 0 (plan + residual) | ✅ |
| Section headers (2, 3.1...) | ✅ |
| Empty spacer rows | ✅ |
| Phase 8 (no work, no plan) | ❌ hidden |
| Everything else | ❌ hidden |

## Правила (16.07.2026)

1. **O — из шаблона, бот не трогает.**
2. **U = O − P** — пересчитывается при генерации ЕЖО.
3. **N = v / O * 100** — реальный % (был хардкод 1=100%).
4. **P = prev_P + v, S = prev_S + v** — кумулятив растёт (было без +v).
5. **Без дубликатов** — строки 655-676 дублировались, исправлено.
6. **prev_P/prev_S** — из yesterday_cum() или шаблона.
