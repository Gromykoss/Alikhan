## Что нужно от заказчика в любом случае

### Обязательно

| Позиция | Описание | Кто делает |
|---------|----------|-----------|
| **WhatsApp-номер** | Отдельный номер телефона для бота. Не личный, не привязанный к другим устройствам | Заказчик покупает SIM, мы настраиваем |

| **WhatsApp-группа «Боевая»** | Рабочая группа стройки (только чтение + погода) | Заказчик создаёт |
| **Данные объекта** | Название, адрес, заказчик, подрядчик, проектировщик, номер договора | Заказчик предоставляет |
| **График СМР** | Этапы, даты начала/окончания (PDF или Excel) | Заказчик предоставляет |
| **Приказы на ИТР** | Приказ о назначении строительного контроля, авторского надзора | Заказчик предоставляет |

## Вариант A: VPS заказчика

### Требования к VPS

| Параметр | Минимум | Рекомендуется |
|----------|---------|---------------|
| **vCPU** | 2 ядра | 4 ядра |
| **RAM** | 4 GB | 8 GB |
| **Диск** | 20 GB SSD | 40 GB SSD |
| **ОС** | Ubuntu 22.04/24.04 | Ubuntu 24.04 |
| **Интернет** | Стабильный, без ограничений по трафику | 100+ Mbit/s |
| **Порты** | 22 (SSH), 3000 (WhatsApp Bridge), 8099 (Extractor) | Только внутренние (не наружу) |
| **Docker** | Установлен (для PostgreSQL) | — |

### API-ключи (предоставляет заказчик)

| Ключ | Назначение | Стоимость | Где получить |
|------|-----------|----------|-------------|
| **xAI API key** | Grok — AI-анализ сообщений и фото | ~$0.30/1M токенов | [console.x.ai](https://console.x.ai) |
| **OpenAI API key** | Codex CLI — само-улучшение кода | ~$20/мес (Pro) или pay-per-token | [platform.openai.com](https://platform.openai.com) |

### Что мы делаем

1. Подключаемся по SSH к VPS заказчика
2. Устанавливаем Docker, PostgreSQL, Python-окружение
3. Клонируем репозиторий Alikhan
4. Настраиваем Hermes Bridge (WhatsApp)
5. Подключаем API-ключи заказчика
6. Инициализируем БД ОЖР под объект
7. Регистрируем WhatsApp-номер через QR
8. Добавляем бота в группы
9. Запускаем, тестируем в песочнице
10. Передаём доступы и документацию

### Ежемесячные расходы заказчика

| Статья | ~Стоимость |
|--------|-----------|
| VPS (4 vCPU, 8 GB) | ~$24-48/мес |
| xAI API (Grok) | ~$10-30/мес (зависит от объёма) |
| OpenAI API (Codex) | ~$20/мес |
| **Итого** | **~$55-100/мес** |

### Что мы предоставляем

| Компонент | Детали |
|-----------|--------|
| **VPS** | 4 vCPU, 8 GB RAM, 40 GB SSD, Ubuntu 24.04 |
| **PostgreSQL** | Docker-контейнер, ежедневные бэкапы |
| **xAI API** | Grok (наш ключ, включён в тариф) |
| **OpenAI API** | Codex CLI (наш ключ, включён в тариф) |
| **Мониторинг** | Uptime-проверки, алерты при падении |
| **Обновления** | Автоматически из GitHub |
| **Поддержка** | Реакция в течение 4 часов в рабочее время |

### Что остаётся за заказчиком

- WhatsApp-номер
- WhatsApp-группы (песочница + боевая)
- Данные объекта и графика СМР
- Приказы на ИТР

### Процесс запуска

1. Заказчик предоставляет данные объекта и график СМР
2. Заказчик покупает SIM, регистрирует WhatsApp
3. Мы настраиваем бота на нашем VPS
4. Заказчик сканирует QR для привязки номера
5. Бот работает → тест в песочнице → боевая группа

### Сроки

| Этап | Время |
|------|-------|
| Предоставление данных заказчиком | 1-3 дня |
| Настройка с нашей стороны | 1 день |
| Тестирование в песочнице | 1-2 дня |
| Запуск в боевую | День 3-4 |

## Часто задаваемые вопросы

### «Почему WhatsApp, а не Telegram?»
WhatsApp — стандарт деловой коммуникации на стройках Кыргызстана и СНГ. Прорабы, ИТР, субподрядчики — все уже там. Не нужно никого переучивать.

### «Нужен ли постоянный доступ в интернет на объекте?»
Боту — да (VPS). Пользователям в WhatsApp — обычный мобильный интернет, как всегда.

### «Что если API ключ закончится?»
Бот пришлёт предупреждение администратору. Работает grace period 24 часа.

### «Можно ли перенести данные с другого объекта?»
Да. База ОЖР — стандартный PostgreSQL. Можно выгрузить, передать заказчику, приобщить к исполнительной документации.

### «Куда сохраняются фото?»
В БД (ojr_photo_log) с привязкой к дате и зданию. Каждое фото получает текстовое описание от Grok vision.
