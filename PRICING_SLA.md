# 💰 Alikhan — Тарифы и SLA

**Дата:** 18 июля 2026 · **Версия:** 1.0

> Два варианта: **«Под ключ»** (мы всё делаем) и **«Self-hosted»** (вы берёте код, мы помогаем запустить).

---

## 1. Тариф «Под ключ» — всё включено

**Для:** генподрядчиков и техзаказчиков, которые хотят готовый AI-инструмент без погружения в DevOps.

### Что входит

| Категория | Состав |
|:----------|:-------|
| **Развёртывание** | Настройка VPS, Docker, PostgreSQL, Hermes Bridge, Python-бота — 1 день |
| **WhatsApp-бот** | Регистрация номера (SIM заказчика), добавление в 2 группы (песочница + боевая) |
| **ОЖР по ГОСТ** | 14 таблиц PostgreSQL (РД-11-05-2007), инициализированных под ваш объект |
| **График СМР** | Импорт графика работ заказчика → `bot_schedule_phases` |
| **VOR-коды** | Импорт справочника видов работ (ВОР) |
| **ЕЖО** | Ежедневное формирование Excel-файла по запросу или авто (cron) |
| **Снимок дня** | Композит: погода, фото, сводка, объёмы — текстовый дайджест |
| **AI-инфраструктура** | Grok (xAI) для QA-парсинга + Grok Vision для описания фото |
| **Бэкапы** | Ежедневный pg_dump, ротация 30 дней, хранение на VPS |
| **Мониторинг** | Watchdog (каждые 5 минут) + health check (8 измерений) + алерты в Discord/Telegram |
| **Хостинг** | VPS 4 vCPU / 8 GB RAM / 40 GB SSD (Ubuntu 24.04) — предоставляем мы |
| **API-ключи** | xAI API + OpenAI API — наши ключи, включены в стоимость |
| **Обновления** | Автоматически из GitHub (основной репо Alikhan) |
| **Поддержка** | Приоритетная: ответ в течение 4 часов в рабочее время (10:00–22:00 UTC+6) |

### Что не входит

- WhatsApp-номер (SIM) — покупает заказчик (~300 сом/мес)
- WhatsApp-группы (песочница + боевая) — создаёт заказчик
- Данные объекта, график СМР, приказы на ИТР — предоставляет заказчик
- Обучение персонала (базовый инструктаж включён, расширенный — отдельно)
- Кастомизация под нестандартные отраслевые требования (обсуждается отдельно)

### Стоимость

| Период | Стоимость | Примечание |
|:-------|:---------|:-----------|
| **Разовый запуск** | **150 000 ₽** | Развёртывание, настройка, импорт данных, тестирование |
| **Ежемесячно** | **45 000 ₽** | VPS, API-ключи, мониторинг, бэкапы, поддержка, обновления |

> 💡 **При оплате за 6 месяцев вперёд — скидка 10%:** 150 000 ₽ + 243 000 ₽ = 393 000 ₽ за полгода.

### SLA

| Параметр | Значение |
|:---------|:---------|
| **Доступность WhatsApp-бота** | 99.5% (не более 3.6 часов простоя в месяц) |
| **Время реакции на инцидент** | ≤ 4 часа в рабочее время (10:00–22:00 UTC+6) |
| **Время восстановления** | ≤ 8 часов (в рабочее время) |
| **RPO (потеря данных)** | ≤ 24 часа (ежедневные бэкапы) |
| **RTO (время до восстановления)** | ≤ 4 часа |
| **Алерты** | Автоматически в Discord/Telegram при падении бота или моста |
| **Обновления** | В течение 48 часов после релиза в GitHub |
| **Поддержка API-ключей** | Авто-пополнение, grace period 24 часа при исчерпании |

## 2. Тариф «Self-hosted» — только код

**Для:** IT-команд подрядчиков, у которых есть свой DevOps и желание контролировать инфраструктуру.

### Что входит

| Категория | Состав |
|:----------|:-------|
| **Исходный код** | Полный репозиторий Alikhan (Python-бот, ОЖР-схема, навыки Hermes, watchdog, бэкапы) |
| **Документация** | AGENTS.md, RUNBOOK.md, README.md, schema.sql, fill guide |
| **Запуск** | Консультация (2 часа) — SSH, Docker, настройка, QR, тест в песочнице |
| **Лицензия** | MIT — свободное использование, модификация, коммерческое применение |

### Что заказчик предоставляет сам

| Компонент | Ориентировочная стоимость |
|:----------|:--------------------------|
| VPS (4 vCPU, 8 GB RAM, 40 GB SSD) | ~$24–48/мес |
| xAI API key (Grok) | ~$10–30/мес (зависит от объёма) |
| OpenAI API key (Codex CLI) | ~$20/мес (Pro) или pay-per-token |
| WhatsApp-номер (SIM) | ~300 сом/мес |
| Администрирование | Силами заказчика |

### Стоимость

| Позиция | Стоимость |
|:--------|:---------|
| **Код + документация** | **90 000 ₽** (единоразово) |
| **Консультация по запуску** | Включено (2 часа) |
| **Дополнительная поддержка** | 5 000 ₽/час |

### SLA (Self-hosted)

| Параметр | Значение |
|:---------|:---------|
| **Код** | Открытый (MIT), полный доступ |
| **Документация** | Актуальная, обновляется вместе с кодом |
| **Обновления** | Доступны в GitHub — заказчик применяет самостоятельно |
| **Консультации** | Ответ в течение 24 часов (email/Telegram) |
| **Бэкапы** | Заказчик настраивает самостоятельно (инструкция в RUNBOOK.md) |
| **Мониторинг** | Заказчик настраивает самостоятельно (watchdog-скрипт включён) |
| **Инциденты** | Заказчик устраняет самостоятельно или заказывает поддержку |

## 4. Часто задаваемые вопросы

### «Можно ли перейти с Self-hosted на Под ключ?»
Да. Вычитаем уже уплаченные 90 000 ₽ из стартового взноса (150 000 − 90 000 = 60 000 ₽). Дальше — ежемесячно 45 000 ₽.

### «Что если объект завершится раньше срока?»
Минимальный срок «Под ключ» — 3 месяца. После этого расторжение за 30 дней.

### «Можно ли клонировать на несколько объектов?»
Да. Для Self-hosted — свободно (лицензия MIT). Для «Под ключ» — каждый дополнительный объект: +25 000 ₽ старт + 25 000 ₽/мес (скидка 45% на второй и последующие).

### «Куда сохраняются данные?»
PostgreSQL на VPS. Ежедневные бэкапы в `/backups/`. Данные принадлежат заказчику — могут быть выгружены в любой момент.

### «Что с безопасностью?»
- WhatsApp-сессия хранится локально на VPS
- API-ключи не коммитятся в Git
- Доступ к БД только с localhost
- Боевая WhatsApp-группа: бот только слушает, без approval сообщения не отправляются
